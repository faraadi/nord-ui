## installation

install using NPM:

````bash
npm install nord-ui
````

or Yarn:

```
yarn add nord-ui
```

## Usage

import component:

```javascript
import 'nord-ui/lib/button.css'
```

import whole package:

```javascript
import 'nord-ui'
```

## installation

install using NPM:

```sh
npm install nord-ui
```

or Yarn:

```sh
yarn add nord-ui
```

## Usage

import component:

```javascript
import 'nord-ui/dist/button.css'
```

import whole package:

```javascript
import 'nord-ui'
```

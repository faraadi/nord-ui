# Quick Start

## Installation

### Package Manager:

```bash
npm install nord-ui
#using Yarn
yarn add nord-ui
```

### CDN:

```html
<link rel="stylesheet" type="text/css" href="https://unpkg.com/nord-ui">
```

## Usage

import a component:

```javascript
import 'nord-ui/lib/button.css'
```

import whole package:

```javascript
import 'nord-ui'
```
